# 🎯 Quiz Application (Windows)

## 📌 Description
This is a Windows-based Quiz Application developed using .NET. It allows users to answer multiple-choice questions and calculates the score based on correct answers.

## 🚀 Features
- Multiple-choice questions  
- Timer-based quiz  
- Score calculation  
- User-friendly interface  

## 🛠️ Technologies
- .NET (Windows Forms)
- C#
